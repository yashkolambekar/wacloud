
Headers are used at certain places like CTA or Templates, these are files which have to be sent with those messages.

Headers can be
- `document`
- `image`
- `video`
- `text`

These are functions which are under the Headers class

## document
| Param | Type   | Description          |
| ----- | ------ | -------------------- |
| link  | string | Link of the document |
## image
| Param | Type   | Description       |
| ----- | ------ | ----------------- |
| link  | string | Link of the image |
## video
| Param | Type   | Description       |
| ----- | ------ | ----------------- |
| link  | string | Link of the video |
## text
| Param     | Type   | Description |
| --------- | ------ | ----------- |
| textValue | string | Text body   |
